## üöÄ Available Commands
Type `help` to see the full list. Key commands include:
* `about`: View bio and principles.
* `skills`: List technical skills (JavaScript, Python, C++, etc.).
* `projects`: See details on projects like the Rock Paper Scissors Game and Simple Calculator.
* `contact`: Display email, phone, and social links.
* `clear`: Clears the terminal history.
